Release 1.0-beta2 – 2012-07-25

Type Changes By
Support for SQL defined in raw strings. So allow def xsql = "SELECT ...". Fixes 558.
Allow different executor types and transaction attributes in SessionManager.
Added typesafe compile time checked inline parameter bindings.
Basic support of Option[_] TypeHandlers. Resolved by JdbcType.
Minor reflection optimizations in ObjectFactory.
All SelectList* now returns Seq[Result] instead of List[Result] to avoid unnecessary collection conversions.
Added suport for iterartive result handlers in Selects. new method handle(ResultContext => Unit)
